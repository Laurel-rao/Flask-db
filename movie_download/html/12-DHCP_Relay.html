12-DHCP_Relay - 答案

实验目的

拓扑与需求

配置与实现

     1. R1 和 SW1 为DHCP中继，向 DHCP_Server 转发DHCP报文，配置静态路由实现网络通信
     2. DHCP_Server 创建地址池，地址池 Produce 为PC1所在区域下发IP地址，网段：192.168.10.0/24，网关为：192.168.10.254，DNS服务器为：114.114.114.114；同时地址池 PC2 为PC2绑定Mac地址，指定获取192.168.20.1，网关为：192.168.20.254，DNS服务器为：114.114.114.114；
     3. PC1 和 PC2 通过DHCP获取IP地址

实验目的

  • 掌握DHCP中继的操作
  • 掌握DHCP绑定Mac地址的操作

拓扑与需求

拓扑：

￼

需求：

1. R1 和 SW1 为DHCP中继，向 DHCP_Server 转发DHCP报文，配置静态路由实现网络通信
2. DHCP_Server 创建地址池，地址池 Produce 为PC1所在区域下发IP地址，网段：192.168.10.0/24，网关为：192.168.10.254，DNS服务器为：114.114.114.114；同时地址池 PC2 为PC2绑定Mac地址，指定获取192.168.20.1，网关为：192.168.20.254，DNS服务器为：114.114.114.114；
3. PC1 和 PC2 通过DHCP获取IP地址

配置与实现

1. R1 和 SW1 为DHCP中继，向 DHCP_Server 转发DHCP报文，配置静态路由实现网络通信

R1

R1(config)#ip route 192.168.10.0 255.255.255.0 12.1.1.2
R1(config)#ip route 192.168.20.0 255.255.255.0 11.1.1.2

R2

R2(config)#ip route 0.0.0.0 0.0.0.0 12.1.1.1

也可配置明细路由

SW1

SW1(config)#ip route 0.0.0.0 0.0.0.0 11.1.1.1

也可配置明细路由

￼

配置DHCP中继

R2

R2(config)#interface ethernet 0/0
R2(config-if)#ip helper-address 10.1.1.1

SW1

SW1(config)#interface vlan 20
SW1(config-if)#ip helper-address 10.1.1.1

2. DHCP_Server 创建地址池，地址池 Produce 为PC1所在区域下发IP地址，网段：192.168.10.0/24，网关为：192.168.10.254，DNS服务器为：114.114.114.114；同时地址池 PC2 为PC2绑定Mac地址，指定获取192.168.20.1，网关为：192.168.20.254，DNS服务器为：114.114.114.114；

DHCP_Server

DHCP_Server(config)#ip dhcp pool Produce
DHCP_Server(dhcp-config)#network 192.168.10.0 /24
DHCP_Server(dhcp-config)#default-router 192.168.10.254
DHCP_Server(dhcp-config)#dns-server 114.114.114.114

DHCP_Server(config)#ip dhcp pool PC2
DHCP_Server(dhcp-config)#host 192.168.20.1 /24
DHCP_Server(dhcp-config)#client-identifier 01aa.bbcc.0006.00
DHCP_Server(dhcp-config)#default-router 192.168.20.254
DHCP_Server(dhcp-config)#dns-server 114.114.114.114

3. PC1 和 PC2 通过DHCP获取IP地址

PC1

PC1(config)#interface ethernet 0/0
PC1(config-if)#ip address dhcp
PC1(config-if)#no shutdown

PC1

PC2(config)#interface ethernet 0/0
PC2(config-if)#ip address dhcp client-id ethernet 0/0
PC2(config-if)#no shutdown
